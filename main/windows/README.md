# Windows Анализаторы

Эта папка предназначена для Windows-специфичных анализаторов.

## Статус

В настоящее время папка пуста. Планируется добавление аналогичных анализаторов для Windows:

- device_analyzer - анализ устройств, батареи и портов
- gpu_monitor - мониторинг GPU
- hardware_rater - оценка производительности
- system_info_lib - библиотека системной информации

## Планируемые модули

### device_analyzer
Анализатор подключенных устройств, батареи и портов для Windows.

**Планируемые функции:**
- Анализ USB устройств через WMI
- Анализ батареи через WMI
- Анализ аудиоустройств через WASAPI
- Анализ портов и типа устройства
- Обнаружение проблем с устройствами

### gpu_monitor
Мониторинг GPU для Windows через DirectX или WMI.

### hardware_rater
Оценка производительности CPU и GPU для Windows.

### system_info_lib
Библиотека системной информации через WMI и Registry.

## Интеграция

При создании Windows-версии анализаторов, они будут иметь аналогичный интерфейс к Linux-версиям для обеспечения кроссплатформенности.
