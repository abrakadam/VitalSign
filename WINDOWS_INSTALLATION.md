# Инструкция по установке VitalSign для Windows

Эта инструкция поможет вам установить и запустить VitalSign на Windows. Даже если вы новичок в программировании, следуйте шагам ниже.

---

## Шаг 1: Установка Python

### 1.1 Скачайте Python

1. Откройте браузер и перейдите на: https://www.python.org/downloads/
2. Нажмите на кнопку **"Download Python 3.12.x"** (последняя версия)
3. Дождитесь окончания загрузки файла `python-3.12.x-amd64.exe`

### 1.2 Установите Python

1. Откройте скачанный файл `python-3.12.x-amd64.exe`
2. **ВАЖНО:** В самом низу установщика поставьте галочку **"Add Python to PATH"**
3. Нажмите **"Install Now"**
4. Дождитесь окончания установки
5. Нажмите **"Close"**

### 1.3 Проверьте установку

1. Нажмите клавиши `Win + R`
2. Введите `cmd` и нажмите Enter
3. В открывшемся окне введите:
   ```
   python --version
   ```
4. Должно появиться сообщение: `Python 3.12.x`

---

## Шаг 2: Скачивание проекта VitalSign

### 2.1 Скачайте архив с GitHub

1. Откройте браузер и перейдите на: https://github.com/abrakadam/VitalSign
2. Нажмите на зеленую кнопку **"Code"**
3. Выберите **"Download ZIP"**
4. Дождитесь окончания загрузки файла `VitalSign-main.zip`

### 2.2 Распакуйте архив

1. Откройте папку "Загрузки" (Downloads)
2. Найдите файл `VitalSign-main.zip`
3. Нажмите правой кнопкой мыши на файл
4. Выберите **"Извлечь все..."** (Extract All...)
5. Нажмите **"Извлечь"** (Extract)
6. Откройте папку `VitalSign-main`

---

## Шаг 3: Установка зависимостей

### 3.1 Откройте терминал в папке проекта

1. Откройте папку `VitalSign-main`
2. Нажмите правой кнопкой мыши на пустое место в папке
3. Удерживая клавишу `Shift`, нажмите правую кнопку мыши
4. Выберите **"Открыть окно PowerShell здесь"** или **"Открыть окно командной строки здесь"**
5. Откроется черное окно (терминал)

### 3.2 Создайте виртуальное окружение

В открывшемся терминале введите:
```
python -m venv venv
```

Нажмите Enter. Подождите несколько секунд.

### 3.3 Активируйте виртуальное окружение

В том же терминале введите:
```
venv\Scripts\activate
```

Нажмите Enter. В начале строки должно появиться `(venv)`.

### 3.4 Установите зависимости

В том же терминале введите:
```
pip install -r requirements.txt
```

Нажмите Enter. Подождите окончания установки (может занять 1-2 минуты).

---

## Шаг 4: Запуск программы

### 4.1 Запуск через терминал

В том же терминале (где `(venv)` в начале строки) введите:
```
python ui/python/gui_main.py
```

Нажмите Enter. Откроется окно программы VitalSign!

### 4.2 Запуск через bat-файл (проще)

1. Откройте папку `scripts` внутри папки `VitalSign-main`
2. Дважды кликните на файл `run_gui.bat`
3. Откроется окно программы VitalSign!

## Возможные проблемы и решения

### Проблема: "python не является внутренней или внешней командой"

**Решение:**
1. Переустановите Python
2. Убедитесь, что галочка **"Add Python to PATH"** была установлена
3. Перезапустите терминал

### Проблема: "Нет модуля PyQt6"

**Решение:**
1. Убедитесь, что виртуальное окружение активировано (должно быть `(venv)` в начале строки)
2. Повторите установку зависимостей:
   ```
   pip install -r requirements.txt
   ```

### Проблема: "Warning: C++ system info library not available"

**Решение:**
Это нормально! Программа будет работать с Python-версией. Не волнуйтесь.

### Проблема: Окно программы сразу закрывается

**Решение:**
Запустите через терминал, чтобы увидеть ошибку:
```
python ui/python/gui_main.py
```

## Обновление программы

Чтобы обновить программу до последней версии:

1. Откройте терминал в папке проекта
2. Активируйте виртуальное окружение:
   ```
   venv\Scripts\activate
   ```
3. Скачайте новую версию с GitHub
4. Замените файлы в папке проекта

---

## Удаление программы

Чтобы удалить VitalSign:

1. Закройте программу
2. Удалите папку `VitalSign-main`
3. Удалите папку `%APPDATA%\VitalSign` (настройки программы)
   - Нажмите `Win + R`
   - Введите `%APPDATA%`
   - Найдите и удалите папку `VitalSign`

## Нужна помощь?

Если у вас возникли проблемы:

1. Проверьте, что Python установлен правильно: `python --version`
2. Проверьте, что виртуальное окружение активировано: должно быть `(venv)` в начале строки
3. Проверьте, что все зависимости установлены: `pip list`
4. Посмотрите проблемы на GitHub: https://github.com/abrakadam/VitalSign/issues
