# Changelog

Все заметные изменения в проекте VitalSign будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено
- Вкладка "Порты и гнезда" для анализа портов ноутбука
- Определение типа устройства (ноутбук/десктоп) через DMI
- Проверка наличия HDMI, USB-C, USB-A, Ethernet, аудиоразъема, SD-слота
- Анализ состояния портов с рекомендациями
- Рекомендации при обнаружении проблем с портами
- Переводы для новой вкладки на русский и английский
- C++ хелперы для оценки CPU и GPU в `helpers/cpp/`
- Python хелпер для веб-ресурсов в `helpers/python/`
- Кроссплатформенная структура с папками `main/linux/` и `main/windows/`
- Определение встроенных видеокарт с корректной оценкой
- Анализ батареи (зарядка, напряжение, ток, мощность, производитель)
- Анализ аудиоустройств (наушники, микрофоны, колонки)
- Анализ устройств ввода (клавиатура, мышь)
- Детальная документация для C++ и Python хелперов

### Изменено
- Реорганизована структура проекта:
  - `main/python/` → `main/linux/`
  - Созданы `helpers/cpp/` и `helpers/python/`
  - Созданы `main/windows/` для будущей Windows-поддержки
- Обновлены все импорты в GUI для новой структуры
- Улучшен механизм перевода интерфейса
- Оптимизирована оценка встроенных GPU (снижение на 40%)

### Исправлено
- Исправлены импорты после реорганизации структуры
- Исправлен путь к web_helper.py
- Исправлены переводы для новых вкладок

## [1.0.0] - 2024-01-XX

### Добавлено
- PyQt6 GUI с современным темным дизайном
- Мониторинг CPU, RAM, диска, сети в реальном времени
- Вкладка "Система" с детальной информацией о BIOS и оборудовании
- Вкладка "Процессы" с сортировкой по CPU и RAM
- Вкладка "Сеть" со статистикой интерфейсов
- Вкладка "Температуры" для мониторинга температуры
- Вкладка "Оценка CPU" с рейтингом процессора
- Вкладка "Оценка GPU" с рейтингом видеокарты
- Вкладка "Драйверы" с информацией о загруженных модулях
- Вкладка "Дистрибутивы" с анализом Linux
- Вкладка "Устройства" с информацией о USB/PCI устройствах
- Вкладка "Помощник" со ссылками на ресурсы
- Система локализации (русский, английский)
- Анимированный фон с огнем
- Настройки приложения
- Кастомные шрифты
- Карточки для отображения информации
- Прогресс-бары для визуализации
- Цветовая индикация нагрузки
- C++ библиотеки для системного мониторинга
- Python анализаторы для Linux

### Изменено
- Полностью переписан GUI на PyQt6
- Улучшен дизайн интерфейса
- Добавлена поддержка множества языков
- Оптимизирован код мониторинга

### Удалено
- Старый GUI на tkinter
- Устаревшие модули
