# Документация GUI интерфейса VitalSign

## Обзор

VitalSign предоставляет два варианта графического интерфейса:
1. **Базовый GUI** (`gui_main.py`) - Простой мониторинг с прогресс-барами
2. **Advanced GUI** (`gui_advanced.py`) - Расширенный мониторинг с графиками

Интерфейс идентичен для Windows и Linux благодаря PyQt6.

## Базовый GUI (gui_main.py)

### Особенности интерфейса

### Темная тема
- Современный темный дизайн для комфортной работы
- Автоматическая адаптация под системную тему

### Вкладки

#### Вкладка "Система"
Содержит следующие секции:

1. **CPU**
   - Прогресс-бар загрузки процессора
   - Цветовая индикация:
     - Зеленый: < 50%
     - Желтый: 50-80%
     - Красный: > 80%
   - Точное значение в процентах

2. **Оперативная память**
   - Прогресс-бар использования RAM
   - Информация об использованном и общем объеме
   - Процент использования

3. **Диск**
   - Прогресс-бар использования дискового пространства
   - Процент заполнения

4. **Сеть**
   - Отправленные данные (MB)
   - Полученные данные (MB)

#### Вкладка "Процессы"
- Таблица топ-20 процессов по загрузке CPU
- Колонки:
  - PID: Идентификатор процесса
  - Имя: Название процесса
  - CPU %: Загрузка процессора
  - RAM %: Использование памяти
- Кнопка "Обновить процессы" для ручного обновления списка

### Автоматическое обновление
- Данные обновляются каждую секунду автоматически
- Фоновый поток не блокирует интерфейс

## Системные требования

### Минимальные
- Python 3.7+
- PyQt6
- psutil

### Рекомендуемые
- Python 3.9+
- 512 MB RAM
- Разрешение экрана 800x600

## Установка и запуск

### Установка зависимостей
```bash
pip install PyQt6 psutil
```

### Запуск
```bash
python gui_main.py
```

Или используйте скрипты:
- Linux: `bash scripts/run_gui.sh`
- Windows: `scripts\run_gui.bat`

## Кроссплатформенность

Интерфейс полностью идентичен на:
- Windows 10/11
- Linux (Ubuntu, Fedora, Debian, etc.)
- macOS (с небольшими отличиями в оформлении)

PyQt6 обеспечивает нативный вид на каждой платформе при сохранении единого дизайна.

## Производительность

- Минимальное использование CPU (< 1%)
- Низкое потребление памяти (~50-100 MB)
- Плавная работа даже на старых системах

## Advanced GUI (gui_advanced.py)

### Дополнительные возможности

#### Графики в реальном времени
- График загрузки CPU с matplotlib
- График использования памяти
- Автоматическое обновление каждую секунду
- Темная тема для графиков

#### История данных
- До 300 точек истории (настраивается)
- Хранение данных CPU, памяти, диска, сети
- Временные метки для каждой точки

#### Панель управления
- Настройка интервала обновления (1-60 секунд)
- Настройка длины истории (10-300 точек)
- Кнопка экспорта в CSV
- Кнопка очистки истории

#### Фильтры процессов
- Фильтрация по CPU
- Фильтрация по памяти
- Сортировка в реальном времени

### Экспорт данных

История данных может быть экспортирована в CSV формат:
- Timestamp (время)
- CPU % (загрузка процессора)
- Memory % (использование памяти)
- Disk % (использование диска)
- Network Sent MB (отправленные данные)
- Network Recv MB (полученные данные)

### Системные требования Advanced GUI

#### Минимальные
- Python 3.7+
- PyQt6
- matplotlib
- numpy
- psutil

#### Рекомендуемые
- Python 3.9+
- 1 GB RAM (для графиков)
- Разрешение экрана 1000x700

### Запуск Advanced GUI

```bash
# Прямой запуск
python gui_advanced.py

# Через скрипты
bash scripts/run_gui_advanced.sh  # Linux
scripts\run_gui_advanced.bat      # Windows
```

### Особенности реализации

- Использует matplotlib для графиков
- Фоновый поток для сбора данных
- Deque для эффективного хранения истории
- QThread для неблокирующего интерфейса
