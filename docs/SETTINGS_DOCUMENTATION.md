# Документация настроек VitalSign

## Обзор

VitalSign предоставляет диалог настроек для конфигурации приложения согласно вашим предпочтениям.

## Доступ к настройкам

Нажмите кнопку **"⚙ Настройки"** в правом верхнем углу главного окна.

## Доступные настройки

### Интервал обновления
- **Описание**: Частота обновления данных системы
- **Диапазон**: 1-60 секунд
- **По умолчанию**: 1 секунда
- **Влияние**: Меньшее значение = более частые обновления, но выше нагрузка на CPU

### Анимация фона
- **Описание**: Показывать или скрывать анимацию огня в фоне
- **По умолчанию**: Включено
- **Влияние**: Отключение может немного снизить нагрузку на GPU

### Количество процессов
- **Описание**: Количество процессов, отображаемых в таблице
- **Диапазон**: 5-50 процессов
- **По умолчанию**: 20 процессов
- **Влияние**: Большее значение = больше данных, но выше нагрузка при обновлении

## Сортировка процессов

Вкладка "Процессы" предоставляет несколько способов сортировки:

### Кнопки сортировки
- **Сортировать по CPU**: Показать процессы с наибольшей загрузкой CPU
- **Сортировать по RAM**: Показать процессы с наибольшим использованием памяти

### Сортировка по клику
- Кликните на заголовок колонки "CPU %" для сортировки по CPU
- Кликните на заголовок колонки "RAM %" для сортировки по памяти

## Цветовая индикация нагрузки

Процессы с высокой нагрузкой подсвечиваются цветом:

### CPU
- **Желтый**: 50-80% нагрузки
- **Красный**: >80% нагрузки

### RAM
- **Желтый**: 50-80% использования
- **Красный**: >80% использования

## Анимации

### Переключение вкладок
При переключении между вкладками "Система" и "Процессы":
- Текущая вкладка плавно исчезает (150ms)
- Новая вкладка плавно появляется (200ms)
- Используется easing функция InOutQuad для плавности

### Фон
- Желто-оранжевый огонь анимируется в фоне
- Частицы поднимаются снизу вверх
- Цвет меняется от желтого к красному по мере поднятия
- Можно отключить в настройках

## Сохранение настроек

Настройки применяются немедленно после нажатия кнопки "Сохранить":

1. Интервал обновления вступает в силу при следующем обновлении
2. Анимация фона включается/выключается мгновенно
3. Количество процессов обновляется сразу в таблице

## Рекомендации

### Для максимальной производительности
- Интервал обновления: 2-5 секунд
- Анимация фона: Отключена
- Количество процессов: 10-15

### Для максимальной информативности
- Интервал обновления: 1 секунда
- Анимация фона: Включена
- Количество процессов: 30-50

### Баланс (рекомендуется)
- Интервал обновления: 1-2 секунды
- Анимация фона: Включена
- Количество процессов: 20

## Сброс настроек

Для сброса настроек к значениям по умолчанию:
1. Откройте диалог настроек
2. Установите желаемые значения
3. Нажмите "Сохранить"

В настоящее время нет функции "Сбросить по умолчанию", но вы можете вручную установить:
- Интервал: 1 секунда
- Анимация: Включена
- Процессы: 20
